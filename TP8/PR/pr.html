Abstracts

O trabalho necessário desenvolver para o sétimo TPC de PRC relaciona-se com o povoamento de uma ontologia definida na aula prática de dia 16 de março de 2020.

Para tal, o professor indicou-nos querys para extrair informação de modo a povoar a ontologia. Foram usadas estas querys, bem como outras definidas pelo aluno, por forma a ter informação ainda mais completa para povoar a ontologia.

O resultado de se terem feito as querys à DBpedia encontram-se nos ficheiros JSON, que se estão na diretoria "resolucao/informacao". A cada um destes ficheiros foi aplicado um programa Python, por forma a converter a informação do formato JSON para TTL. Os programas encontram-se na diretoria "resolucao/conversao". O resultado de cada uma destas conversões encontra-se na diretoria "resolucao/resultadoConversao". Cada um destes resultados foi adicionado à ontologia definida na aula prática do dia 16 de março de 2020, ou seja, ao ficheiro "aula7.ttl", dando origem à ontologia final, que se encontra no ficheiro "ontologia.ttl". Cada um dos resultados referidos anteriormente é exemplificado da seguinte forma: ao ficheiro "atores.json" que se encontra na diretoria "resolucao/informacao" foi aplicado o programa python "conversorAtores.py, que se encontra na diretoria "resolucao/conversao", com o uso do comando "python conversorAtores.py > atores.ttl". O conteúdo do ficheiro "atores.ttl" foi adicionado à ontologia definida na aula prática. O mesmo processo se aplicou para escritores, realizadores e países. Ao ficheiro JSON que contem os filmes(ficheiro "filmes.json" que se encontra na diretoria "resolucao/informacao") foram geradas, para além dos filmes, as línguas referentes a cada país. Esta informação encontra-se no ficheiro "filmesELinguas.ttl", na diretoria "resolucao/resultadoConversao"

Importante referir que os ficheiros JSON que resultaram das querys efetuadas à DBpedia sofreram alterações no seu conteúdo, para facilitar o trabalho que era necessário fazer pelos programas python.
