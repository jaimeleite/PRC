Abstracts

O trabalho necessário desenvolver para o oitavo TPC de PRC relaciona-se com a definição de um conjunto de rotas para ser possível obter informação presente num repositório GraphDB, sobre cinema.

De seguida, apresentam-se as rotas que foram tratadas: "/filmes", sendo possível visualizar a listagem dos filmes presentes no repositório; "/filmes/:id", sendo indicadas as informações relativas a um filme específico; "/atores", sendo apresentados os atores registados no repositório; "/atores/:id", em que são apresentadas informações para um ator específico; "/personagens", sendo indicada a lista com todas as personagens; "personagens/:id", em que é apresentada informação sobre uma personagem específica; "/paises", sendo apresentados todos os países; "/linguas", fazendo aparecer todas as línguas existentes no repositório; "/generos", sendo possível visualizar os géneros que estão presentes no repositório.

Importante referir que foram definidos seis ficheiros de route, para além do index.js que é criado automaticamente aquando da criação da aplicação. Consequentemente, foram criados seis ficheiros, que se encontram na pasta controllers, onde estão presentes as queries que são necessárias efetuar para cada rota.

O processo a efetuar para as rotas "/filmes" e "/filmes/:id" encontra-se no ficheiro filmes.js, na pasta routes. Ainda nesta pasta, estão presentes os ficheiros: atores.js, onde se tratam as rotas "/atores" e "/atores/:id"; generos.js, onde é tratada a rota "/generos"; linguas.js, onde é tratada a rota "/linguas"; paises.js, sendo possível tratar a rota "/paises"; personagens.js, onde são tratas as rotas "/personagens" e "/personagens/:id". Na pasta controllers, os ficheiros que têm nome igual aos que se encontram na pasta routes contêm as queries necessárias para devolver a informação para as rotas contidas nos ficheiros da pasta routes.
