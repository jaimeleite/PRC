Abstracts

O trabalho que foi necessário efetuar para o terceiro TPC de PRC relaciona-se com o desenvolvimento de um servidor em NodeJs que permite ao utilizador executar uma query em SPARQL, relativa aos dados armazenados num dado repositório que terá de indicar. O servidor em NodeJs, que se encontra na pasta "App", apresenta ao utilizador os repositórios que este tem disponíveis no servidor do GraphDB. Assim, tem à sua disposição duas textArea, uma para indicar o nome do repositório e outra para indicar a query em SPARQL. Quando o utilizador carrega no botão Exec, é enviado um pedido ao servidor do GraphDB que retorna valores que são armazenados numa outra textArea. O conteúdo desta pode ser removido se o utilizador pressionar o botão Limpar.
