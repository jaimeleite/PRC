Abstracts

O trabalho necessário desenvolver para o sexto TPC de PRC relaciona-se com a resolução de um texto de um dos anos anteriores da unidade curricular PRC. Neste teste, é requerida a realização de quatro exercícios.

O primeiro deles é a obtenção de um grafo, o qual se encontra na pasta "TP6/Exs/Exercicio1", nomeadamente no ficheiro que tem o nome de result.png. Para a obtenção deste grafo, a ontologia apresentada para este exercício foi guardada num ficheiro e este foi carregado no GraphDB, onde foi possível visualizar as relações entre os nodos.

O segundo exercício relaciona-se com a criação de uma ontologia, partindo dos parâmetros indicados no enunciado para este exercício. A ontologia foi desenvolvida, tendo por base os conhecimentos aprendidos nas aulas práticas de PRC. A ontologia encontra-se definida na pasta "TP6/Exs/Exercicio2", nomeadamente no ficheiro ex2.ttl. No exercício 2, era requerido que se desenvolvessem queries SPARQL. Estas queries encontram-se na diretoria "TP6/Exs/Exercicio2", no ficheiro sparql.txt.

O terceiro exercício consistia em responder a um conjunto de queries SPARQLcom a informação presente na DBpedia. As queries encontram-se na pasta "TP6/Exs/Exercicio3", nomeadamente no ficheiro resolucao.txt.

O quarto exercício consistia em, partindo de uma ontologia sobre Pokemons, definir um conjunto de queries SPARQL relativas a essa mesma ontologia. A ontologia foi guardada num ficheiro, designado por pokemon.rdf, na diretoria "TP6/Exs/Exercicio4". As queries encontram-se na pasta "TP6/Exs/Exercicio4", nomeadamente no ficheiro resolucao.txt.
