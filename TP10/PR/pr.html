Abstracts

O trabalho necessário desenvolver para o décimo TPC de PRC relaciona-se com a definição de queries SPARQL para obter de uma ontologia armazenada num repositório GraphDB informação acerca de: ancestrais, descendentes, sogros, cunhados, tios e primos. No ficheiro "queries.txt", na pasta "resolucao", encontram-se definidas queries para obtenção da informação referida anteriormente, bem como queries construct para se formarem as relações de "hasAncestor", "hasDescendent", "hasParentInLaw", "hasSiblingInLaw", "hasUncleAunt" e de "hasCousin".
