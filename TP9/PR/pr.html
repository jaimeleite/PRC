Abstracts

O trabalho necessário desenvolver para o nono TPC de PRC relaciona-se com a definição de um conjunto de rotas num servidor de API de dados, bem como a continuação do desenvolviemto de uma interface que permita visualizar a informação contida num repositório GraphDB, sobre cinema.

De seguida, apresentam-se as rotas que foram tratadas no servidor de API de dados: "/atores", sendo possível obter a listagem dos atores presentes no repositório; "/atores/:idAtor", sendo indicadas as informações relativas a um ator específico; "/personagens", sendo apresentadas todas as personagens registadas no repositório; "/personagens/:idPersonagem", sendo apresentados os dados relativos a uma personagem em específico.

Importante referir que, neste servidor, as rotas mencionadas anteriormente foram definidas no ficheiro "index.js", contido na pasta "routes". As queries SPARQL necessárias efetuar para se obter a informação do GraphDB relativa às rotas dos atores encontram-se no ficheiro "atores.js", na pasta "controllers". Nesta pasta encontram-se também, no ficheiro "filmes.js", as queries necessárias para obter do GraphDB informação relativa às personagens.

No servidor de interface, é possível navegar entre filmes, atores, personagens, sendo apresentados os respetivos atributos para cada um destes.
